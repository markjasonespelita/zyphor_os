Zyphor OS
Version 1.0.0

What's New

First stable release — built from the ground up.

🐚 zyshell — Custom Shell
Arrow-key history navigation

Browse previous commands using the up and down arrow keys.

I/O redirection

Full support for >, >>, and < operators.

Pipeline support

Chain commands together using | pipes.

Git branch in prompt

Displays the current Git branch name directly in the shell prompt.

Glob & tilde expansion

Supports ~ home expansion and wildcard glob patterns.

Color ls output

File listings are color-coded by type automatically.

🖥️ Desktop & GUI
SDL2 GUI Framework

Custom widget library with buttons, headings, text inputs, and word-wrapped paragraphs.

Grid layout system

Bootstrap-style responsive grid built on top of SDL2 and SDL2_ttf.

What's New viewer

This screen — launches once after each system update to show release notes.

📦 System & Utilities
C helper libraries

Built-in string, validation, and command utilities available system-wide.

JSON support

Lightweight C JSON library with read, push, delete, and update operations — no third-party dependencies.

.deb package manager integration

All Zyphor OS components ship as installable .deb packages.
