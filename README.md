# Zyphor OS Legacy Repository

This repository is the original home of Zyphor OS.

Development has since been transferred to the official [Zyphor OS](https://github.com/zyphor-os) GitHub account and its new repository. This repository is no longer the primary source for active development.

Rather than removing it, we have chosen to preserve this repository as part of the project's history. It contains the early work, experiments, milestones, and progress that helped shape Zyphor OS from its very first day.

This repository will remain available as a historical archive, documenting the journey of Zyphor OS from Day One.

Thank you to everyone who contributed, tested, learned, and grew with the project.

— The Zyphor OS Team

Zyphor OS Desktop - https://github.com/zyphor-os/zyphor-os-desktop

Zyphor OS Minimal - https://github.com/zyphor-os/zyphor-os-minimal  
Zyphor Website and Package Repository - https://github.com/zyphor-os/zyphor-os.github.io
